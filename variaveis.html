Variáveis e Tipos de Dados

Para declarar variáveis, usamos o comando var e informamos o identifcador. Não é preciso especificar o tipo de dado que a Variável irá receber, pois o JS atribui o tipo de forma dinamica.

É utilizado o sinal de "=" para representar que uma Variável RECEBE o valor informado.

        
        var nome = 'Rodrigo' //Variável do tipo String(Textos)
        var idade = 18 // Variável do tipo Número Inteiro
        var trabalhando = true //Variável do tipo boolean(verdadeiro ou falso)
        var salario = 5000.50 //Variável do tipo Número Real

Agora, vá na parte de script do seu documento HTML e crie as suas variáveis com seus valores, e utilize o comando window.alert()para exibir os valores no navegador.
