Estrutura Condicional

Condição Simples

Aprendemos até agora, formas de programar de forma sequencial. O código é lido de cima pra baixo, da esquerda para a direita, de forma que ele não possui nenhum desvio de caminho.

Porém, na vida, sempre é preciso fazer escolhas e tomar caminhos diferentes do que apenas seguir em frente.

Para isso é usado o comando if. Em JavaScript ele é utilidado da seguinte maneira:

        
        if (idade > 18) {
            window.alert('Maior de Idade')
        }

Desta maneira, o programa só executará o comando window.alert('Maior de Idade') apenas quando o valor dentro da variável idade for maior que 18.

Neste caso, estamos apenas retornando algo quando a idade > 18

Condições Compostas/Aninhadas

E se houver mais condições a serem respeitadas e situações possíveis de acontecer? Simples, usaremos então as chamadas 'Condições Compostas', aonde realizaremos o teste da condição e, caso ela não seja verdade, faremos um novo teste para uma outra condição.
